03報價與費用配置

技術開發勞務費用 TWD 15,000

此為乙方技術執行費用（含人設建模、系統開發、安全防護），不包含第三方平台支出。

甲方負擔之實支實付項目

  • LLM API 費用 — 依展覽期間觀眾對話量計費，實際金額視模型選擇與使用量而定
  • 雲端主機 — 視需求而定（部分平台如 Vercel、Railway 等有免費或低成本方案）
  • 網域 — 若需自訂網域，由甲方購買

04帳號權限與費用支付方式

採「甲方持有帳號，乙方持有開發權限」模式，確保費用透明、帳單歸屬明確：

LLM API 帳號設定

  1. 甲方以官方 Email 註冊 API 帳號（如 Anthropic / OpenAI）
  2. 於 Billing 頁面綁定甲方信用卡，帳單直接歸屬甲方
  3. 設定月度預算上限，超額自動停止，避免意外費用
  4. 於 Members / IAM 頁面邀請乙方 Email 為 Developer，乙方取得 API Key 管理權限

雲端平台帳號設定

  1. 甲方以展覽單位名義註冊雲端平台帳號（如 Vercel、Railway、Cloudflare 等）
  2. 將乙方設為 Collaborator / Admin，由乙方負責程式碼部署
  3. 展覽結束後，乙方移除自身權限，帳號完整歸還甲方
為什麼這樣做？ 乙方不經手任何金流，甲方可隨時在後台查看用量與帳單。展後所有帳號、程式碼、資料皆歸甲方所有。

05預期時程

本案含 6 次討論，每次討論後交付可操作的 prototype：

  • 開發期 每週 1 次固定討論，中間以非同步訊息溝通（乙方每日統一回覆一次）
  • 收尾期 上線前一週可加密討論頻率，快速迭代細節與修正
  • 交付 三月底前正式上線佈署
